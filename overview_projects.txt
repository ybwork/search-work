1. Личный кабинет жителя коттеджного посёлка.

    Инструменты: PHP, Laravel, MySQL (впоследствии переписан на Python + Django + PostgreSQL)
    
    Функционал:

        - Оплата ЖКУ

        - Заказ дополнительных услуг

        - Взаимодействие с пропускным контролем. (Интеграция с системой автоматического открытия ворот "Автомаршалл")

        - Интеграция с 1С для получения всей необходимой информации по текущему счёту и его истории

        - Интеграция с API чек онлайн

        - Интерфейс для администратора в котором он может работать с пользователями (CRUD) и видеть все их действия.

    Роль: back-end

2. Платформа для продвижения объектов недвижимости.

    Инструменты: PHP, MySQL (впоследствии переписан на Python + Django REST Framework + PostgreSQL)
    
    Функционал:

        - Панель администратора с возможностью создать объект недвижимости и наполнить его квартирами, планировками, ценами и другой информацией.

        - Интерфейс менеджера для демонстрации и бронирования квартир.

        - Интеграция с API AmoCRM

    Роль: back-end

3. Платформа для торгов в режиме реального времени.

    Инструменты: PHP, MySQL, Socket.IO, Redis.

    Функционал:

        - Панель администратора, где он создаёт лот и выставляет его на торги.

        - Интерфейс пользователя, где он может делать ставку на выставленный лот.

        - Все действия пользователей в процессе аукциона работают в режиме real time.

    Роль: back-end

4. Личный кабинет для пункта обмена валют.

    Инструменты: Python, Django, PostgreSQL, Celery, Redis.

    Функционал:

        - Панель администратора, где он создаёт валюту и курс.

        - Интерфейс пользователя, где он может посмотреть текущие курсы.

        - В дальнейшем курс валют обновляется один раз в сутки с помощью Celery и Redis.

    Роль: back-end
